MBA AI Agent Course Optimization Plan (Expanded)

Based on Stanford CS146S (Fall 2025) Curriculum Reference: https://themodernsoftware.dev/

1. Executive Summary

This expanded plan incorporates insights not just from the syllabus, but from the detailed Week 1 Lecture Slides and the Comprehensive Reading List. The goal is to shift the MBA course from “LLM Theory” to “Modern Software Developer Reality” — emphasizing tooling, context engineering, and the full AI lifecycle.

2. Structural & Content Upgrades

Module A: The AI Developer’s Mindset (Lessons 1-2)

Optimization Source: Week 1 Slides “Introduction and How LLMs are Made”

  • New Concept: “State of the World 2025”
    • Visualize: “You won’t be replaced by AI, but by an engineer using AI.”
    • Metric: Software development efficiency (10x potential).
  • Revised Theory: “3 Stages of Training” (Pretraining -> SFT -> RLHF/Preferences)
    • Replace generic “training” slides with this specific 3-stage pipeline from Stanford slides.
    • Highlight “Preferencing tuning” (RLHF) as the key to alignment.
  • New Insight: “Not All Tokens Are Created Equal”
    • Discuss “Middle-loss” in context windows (from reading list “How Long Contexts Fail”).
    • Practical tip: Put critical instructions at the beginning and end.

Module B: The Tooling Ecosystem (Lessons 3-6)

Optimization Source: Week 2-5 Syllabus & Readings

  • New Lesson: “The AI IDE: Cursor & Windsurf”
    • Reading: “Specs Are the New Source Code” (Ravi Mehta).
    • Lab: Setting up .cursorrules for project-specific context.
  • New Lesson: “The Modern Terminal: Warp”
    • Reading: “Warp University”.
    • Activity: Using natural language to generate shell scripts in Warp.
  • Deep Dive: “Claude Code Anatomy”
    • Source: “How Anthropic Uses Claude Code” (PDF).
    • Key Concept: System prompts, “System Reminders” ( tags), and sub-agent spawning.

Module C: Building & Security (Lessons 7-10)

Optimization Source: Week 6 & 9 Syllabus

  • New Lesson: “AI Security (SAST/DAST)”
    • Reading: “Agentic AI Threats” (Unit 42) & “Copilot RCE”.
    • Concepts: Prompt injection, package hallucination, “secure vibe coding”.
    • Guest Speaker Profile: Reference Isaac Evans (Semgrep).
  • New Lesson: “Agent DevOps”
    • Reading: “Context Rot” (Chroma Research).
    • Activity: Tracing agent execution loops (LangSmith/Arize).
