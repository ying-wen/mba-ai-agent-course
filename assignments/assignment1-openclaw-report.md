# Assignment 1: OpenClaw 使用报告

## 🎯 作业目标
亲手完成 OpenClaw 的安装部署与核心功能体验，建立对「Agent 如何通过工具与 Skills 在真实世界执行任务」的直观理解。

---

## ✅ 作业要求

### 1) 安装部署 OpenClaw
- 在自己的电脑或云主机完成 OpenClaw 安装与运行
- 能够启动网关/服务，并成功进行至少一次对话交互

### 2) 体验核心功能
请至少覆盖以下三类能力：

- **消息集成**: 任选渠道完成接入与对话
- **工具调用**: 让 Agent 实际调用至少 2 次工具
- **Skills 机制**: 安装/启用至少 2 个 Skills

### 3) 输出使用报告
建议包含以下结构：

1. **环境与安装过程**
2. **功能体验记录**
3. **问题清单（Issue Log）**
4. **改进建议**
